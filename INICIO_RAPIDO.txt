PASSO 1: VERIFICAR PRÉ-REQUISITOS
═════════════════════════════════
Execute: VERIFICAR_SISTEMA.bat

Este script verifica se você tem:
✓ Node.js instalado
✓ Python instalado
✓ MySQL rodando
✓ Banco de dados configurado

─────────────────────────────────────────────────────────────────

PASSO 2: CONFIGURAR BANCO DE DADOS (Primeira vez)
═════════════════════════════════════════════════
1. Abra o XAMPP Control Panel
2. Clique em "Start" no MySQL
3. Execute: SETUP_DATABASE.bat

Isso criará:
✓ Banco de dados helixai_db
✓ Tabelas cars e vendors

─────────────────────────────────────────────────────────────────

PASSO 3: INSTALAR DEPENDÊNCIAS (Primeira vez)
══════════════════════════════════════════════
Execute: INSTALAR_DEPENDENCIAS.bat

Isso instalará todas as dependências de:
✓ Backend Flask (Python)
✓ Landing Page (Node.js)
✓ CRM Cliente (Node.js)
✓ CRM Admin (Node.js)
✓ WhatsApp Service (Node.js)
✓ AIra Auto (Node.js)
✓ AIra Imob (Node.js)

─────────────────────────────────────────────────────────────────

PASSO 4: INICIAR O SISTEMA
═══════════════════════════
Execute: INICIAR_TUDO_V2.bat

Isso iniciará automaticamente:
1. Backend Flask API (porta 5000)
2. Landing Page (porta 5176)
3. CRM Cliente (porta 5177)
4. CRM Admin (porta 5175)
5. WhatsApp Service
6. AIra Auto (Bot Veículos)
7. AIra Imob (Bot Imóveis)
