Importe seu catálogo de produtos para que o robô de atendimento possa apresentá-los aos clientes automaticamente.

Formato do CSV:

Seu arquivo deve conter as seguintes colunas:

nome,descricao,categoria,preco,preco_promocional,estoque,sku,marca,disponivel,palavras_chave,link,imagem_url

Como funciona?
  1. Baixe o template CSV acima para ver o formato correto
  2. Preencha com seus produtos usando Excel, Google Sheets ou editor de texto
  3. Salve como CSV (separado por vírgulas)
  4. Faça o upload usando o formulário acima
  5. Pronto! O robô já terá acesso ao seu catálogo completo
Dicas importantes:
  • O campo nome é obrigatório
  • Use true ou false para o campo disponivel
  • Preços devem usar ponto (.) como separador decimal: 59.90
  • Palavras-chave podem ser separadas por vírgula: camiseta,básica,preta
